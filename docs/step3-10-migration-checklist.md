# Step3~10 Migration Checklist

Step3~Step10 전환에서 화면 일치성과 기능 흐름을 함께 검증하는 체크리스트입니다.

## 1) 구조 점검
- [ ] `Step3.tsx`~`Step10.tsx`에서 `LegacyHtmlStep` 참조가 제거되었는지 확인
- [ ] `step3`~`step10` 하위에 `StepXScreen.tsx`가 생성되었는지 확인
- [ ] `npm run build`가 성공하는지 확인

## 2) 단계별 기능 점검
- [ ] Step3: 주제 2개 이상 + 활동 1개 선택 후 다음 이동
- [ ] Step4: 약속 체크 전 시작 버튼 비활성
- [ ] Step5: 100자 제한/카운터/빈값 차단
- [ ] Step6: 내 답변 + 상대 답변 렌더링
- [ ] Step7: `다음`은 8단계, `바로 끝내기`는 10단계 이동
- [ ] Step8: 감정 선택 진행 + 건너뛰기 둘 다 작동
- [ ] Step9: 문장 선택 후 저장 및 다음 이동
- [ ] Step10: 미션 선택 후 완료(`/done`) 이동

## 3) 상태 복구 점검
- [ ] 각 단계 입력 후 새로고침해도 선택값이 유지되는지 확인
- [ ] Step7에서 10단계 점프 후 완료까지 상태가 이어지는지 확인

## 4) 완료 점검
- [ ] `/done`에서 Step1 닉네임, Step8 감정, Step9 문장, Step10 미션 요약이 보이는지 확인
