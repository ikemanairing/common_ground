## 2) 진입 / 복구
- [ ] `/` 진입 시 새 세션은 Step1에서 시작된다.
- [ ] Step1 진행 후 새로고침하면 예상 경로로 복구된다.
- [ ] 중간 단계에서 직접 URL 진입 시 허용된 단계는 열리고, 허용되지 않은 단계는 현재 진행 위치로 리다이렉트된다.

## 3) 질문 흐름
- [ ] Step1 완료 후 Profile -> Interests -> Topics -> Promise -> Q1 흐름이 자연스럽다.
- [ ] Q2~Q8은 이전 질문 답변이 없는 상태에서 건너뛸 수 없다.
- [ ] Q1~Q8 답변 후 Compare로 이동한다.

## 4) 분기 흐름
- [ ] Wrap-up에서 `다음`을 누르면 Emotion으로 이동한다.
- [ ] Wrap-up에서 `바로 끝내기`를 누르면 Mission으로 바로 이동한다.
- [ ] `바로 끝내기` 분기에서 Mission의 뒤로가기는 Summary가 아니라 Wrap-up으로 돌아간다.

## 5) 완료 흐름
- [ ] Emotion 선택 후 Summary로 이동한다.
- [ ] Emotion 건너뛰기도 정상 동작한다.
- [ ] Summary 선택 후 Mission으로 이동한다.
- [ ] Mission 완료 후 Done으로 이동한다.
- [ ] Done 화면에 닉네임, 감정, 오늘의 문장, 내일 미션이 기대한 값으로 표시된다.

## 6) 기록
- [ ] 실패한 시나리오와 재현 절차 기록
- [ ] 수정 필요 UX 항목 기록
- [ ] 후속 작업을 `Todo.md` 또는 관련 문서에 반영
